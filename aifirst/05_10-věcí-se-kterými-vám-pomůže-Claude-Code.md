# 10 věcí, se kterými vám pomůže Claude Code

Vibe coding hýbe nejenom světem studentů méhokurzu AI First. Myslím si, že je to jeden znejsilnějších trendů, který se aktuálně děje v San Franciscu.

Claude Code je pak za mě ten nejlepší způsob, jak z vibe codingu vytěžit maximum. Jak se naučit vibe codovat udržitelně a mít prakticky nekonečné možnosti, co lze vytvořit.

A právě trochu inspirace, proč se učitClaude Codea co se s ním dá všechno dělat, vám chci dát v tomto seznamu. Samozřejmě můžeme „generovat“ zdrojový kód webových stránek a aplikací. Ale Claude Code se dá využít univerzálně jako AI agent nainstalovaný v počítači na mnohem více. Prakticky na téměř cokoliv.

Tady je 10 věcí, které můžete dělat v Claude Code:

Vyrábět jednoduché webové hry

Testovat bezpečnost webů a aplikací

Vyrábět 3D hry

Třídit soubory na vašem počítači

Tvořit automatizace v N8N

Přeložit celý váš web do jiného jazyka

Snížit rozlišení všech fotek ve složce

Scraping dat z internetu a práce s nimi

Tvorba 3D grafiky textem

Stahovat data z API Google Analytics, Search Console a lépe je vizualizovat

1. Vyrábět jednoduché webové hry

Ideální pro začátek. Nebo pro to, abyste mohli přivést k vibe codingu děti. Můžete vytvářet adaptace hry Snake – tady např. můjplzeňský had se třemi soupeři, varianty Pacmana, tetrisu, sudoku, skákačky jako třebaUtíkej Kančileod Honzy Páva.

2. Testovat bezpečnost webů a aplikací

Zkuste zabudovaný příkaz /security-review přímo v Claude Code. Mnohem pokročilejší způsob je využitíMCP Semgrep, které si připojíte a pomůže vám skenovat váš zdrojový kód pro tisíce nejběžnějších chyb a hrozeb. Nebo sami požádejte Claude Code o kontrolu specifických zranitelností. Je ale mnohem lepší si na to dát pozor hned na začátku než odstraňovat chyby na konci vývoje.

3. Vyrábět 3D hry

Zkuste si vytvořit vaši první webovou 3D hru ve frameworkuthree.js. Já jsem takhle dělal 14 dní na svém walking simulátoru Brna:HrejBrno.cza naučil jsem se díky tomu o vibe codingu a technologiích snad nejvíc za celý rok. A navíc to byla ohromná zábava.

4. Třídit soubory na vašem počítači

Claude Code je oproti čistě webovým appkámLovable,V0neboReplitplnohodnotný kódovací agent, který funguje v konzoli vašeho počítače. Může tak, pokud mu to dovolíte, přistupovat k vašim souborům.

Třeba mu řeknete: Projdi všechny faktury v této složce, zjisti k jaké patří firmě a utřiď je do složek podle dodavatelů a přejmenuj do formátu: cislofaktury_nazevdodavatele_datum.pdf

5. Tvořit automatizace v N8N

Mrkněte naN8N MCP, díky kterému vám Claude Code bude schopen navrhnout automatizační schémata přímo ve vašem N8N účtu. Feels like magic! Pokud je vám úplně volné soukromí a bezpečnost, můžete o výrobu vaší automatizace požádat ChatGPT Atlas. Ten to zvládne ještě rychleji. Není to dokonalé, ale skutečně se touto cestou dá tvořit.

6. Přeložit celý váš web do jiného jazyka

Stáhněte si celý web z FTP do vašeho počítače. Necháte Claude Code ať se v kódu vyzná.

Na to stačí pro začátek i příkaz /init.

A pak ho požádáte, ať si udělá todo list a pečlivě přeloží každou stránku po stránce a pohlídá si kontext, o čem ten web je. Potom si můžete udělat třeba další instanci Claude Code, který bude něco jako jazykový korektor a celé to po tom prvním zkontroluje.Více o tom píšu tady.

7. Snížit rozlišení všech fotek ve složce

Zadejte Claude Code.

Udělej si skript, který sníží rozlišení všech fotek ve složce, aby měli delší stranu max 1000 px, byly ve formátu .JPEG a měli max 100 KB.

A hotovo. Bez dalšího stahování software nebo zkoumání co a jak. Tenhle způsob práce lze aplikovat skoro na cokoliv.

8. Scraping dat z internetu a práce s nimi

Tuhle případovkujsem vám ukazoval nedávno. Claude Code bez problémů stáhne třeba data o 900 přednášek konference, na kterou jedete. Uloží do databáze. Hotovo raz dva. A pak třeba může poradit, která z těch akcí pro vás bude nejvíce přínosná. Tak vypadá AI First mindset.

9. Tvorba 3D grafiky textem

Zkuste si spustitBlender MCP server. Fascinující věc. Hraju si s tím a přemýšlím, že udělám remaster své hry HrejBrno :))

10. Stahovat data z API Google Analytics, Search Console a lépe je vizualizovat

Jen co vaše představivost stačí :)) Claude Code si dokáže data stáhnout, a pak si třeba napsat automatizaci pro N8N, aby si je stahoval pravidelně každý den. Grafy pak můžete navrhnout přesně na míru vám nebo vašim kolegům.

Závěr

Vibe coding a konkrétně Claude Code mnohonásobně přesahuje to, co si většina lidí představí: programování přes AI. Je to nový způsob přemýšlení, nový způsob práce. Za měnejužitečnější dovednost dneška.

Chcete se to naučit? Tenhle způsob myšlení a Claude Code učím krok za krokemve svém kurzu.
